# Good Outcomes Skills Marketplace

A Claude Code plugin marketplace for Good Outcomes career applications and tools.

## Installation

### Option 1: Via skills.sh (Recommended)

The fastest way to install:

```bash
npx skills add DataEQ/go-people-skills
```

This works with Claude, Codex, Cursor, Moltbot, and any other harness that supports skills.

### Option 2: Claude Code Plugin Marketplace

Add the marketplace and install the plugin:

```
/plugin marketplace add DataEQ/go-people-skills
/plugin install apply-to-good-outcomes@go-people
```

### Option 3: Direct Clone

```bash
git clone https://github.com/DataEQ/go-people-skills.git ~/.claude/plugins/go-people-skills
```

## Usage

Once installed, start your application:

```
/apply
```

This guides you through:
1. Sharing your background and experience
2. Solving a semantic embedding challenge
3. Uploading your resume (optional)
4. Submitting your application

The challenge tests your ability to work with AI tools, research, and iterate—core skills for the roles.
